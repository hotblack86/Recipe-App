**A Recipe App built with React**

*Prerequisites*

- Have Node and NPM installed

- Sign up for EDAMAM account

- In App.js set APP_ID and APP_KEY to the API id and key strings given to you by EDAMAM


**Use**

- Run the App 
```
npm start
```
